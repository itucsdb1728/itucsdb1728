Download IBM DB2 Express-C for Linux 32 bit from:

http://www.ibm.com/software/data/db2/express-c/

Put the *_expc.tar.gz file into the directory itucsdb/db2.

For DB2 installation on Vagrant, go to itucsdb/db2 and run: vagrant up

Local credentials: localhost, 50000, vagrant, vagrant, itucsdb

Add "SQL Database (IBM)" service (not "IBM DB2 on Cloud") to Bluemix.

Info about using Python with DB2:

http://www-01.ibm.com/support/knowledgecenter/SSEPGG_10.1.0/com.ibm.swg.im.dbclient.python.doc/doc/c0054366.html
